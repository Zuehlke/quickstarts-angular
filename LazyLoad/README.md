LazyLoad Quickstart
==================

This quickstart demonstrates how to dynamicly load modules into an angular application.
This quickstar shows:
 - How to dynamicly resolve a controller for a view (the shop) and how this is integrated in ui.router
 - How to dynamicly load a complete module
 - How to store settings in the locale storage

## Notes

The sample is build with VisualStudio and WebEssentials. The code is written in TypeScript.
Dependencies are managed with nuget.
