Handling modules with Angular and ocLazyload

This sample demonstrates how to lazy load controllers or modules in angular.

  • Used technologies:

    • Angular
    • ocLazyload
    • Angular Ui Route
    • LocalStorageModule

    The shop controller is loaded when the ui-router navigates to the shop page.

    • The ui-router is configured to resolve a promise when the shop state is set.
    • The ocLazyload is configured to load the file with the browser integrated script loader

    The radiator module is loaded after the buy button is clicked.

    • The radiator module adds a new entry to the nav-bar.
    • The radiator module adds a new state to the ui-router.
    • The page is reloaded if the module gets uninstalled.

    The installed items are stored in the local storage of the browser.
